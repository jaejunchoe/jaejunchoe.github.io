# Abstract
본 연구는 재생에너지 보급 확대에 따른 원자력발전의 유연한 출력 조절 요구에 대응하기 위해, 강화학습(Reinforcement Learning)을 활용한 i-SMR(innovative Small Modular Reactor) 부하추종 운전 최적화 알고리즘을 개발하였다. Flownex SE 시뮬레이션 환경과 Python API를 연동하여 4개 모듈의 SMR 운전을 동시에 제어하는 SAC(Soft Actor-Critic) 기반 에이전트를 구현하였다. 44개의 상태 변수와 12개의 행동 변수를 정의하고, 다수 모듈 간 균형을 고려한 보상 함수를 설계하여 1,000회 에피소드 학습을 수행하였다. 실험 결과, 재생에너지 변동에 대응한 부하추종 운전 가능성을 검증하였다.

# Research Background
- 원자력발전은 기저부하전원으로서 핵심 역할을 수행해왔으나, 재생에너지 보급 확대로 유연한 출력 조절 능력이 요구됨
- 부하추종운전은 전력계통의 안전성과 경제성을 동시에 확보하기 위한 주요 운전모드
- i-SMR은 안전성, 경제성, 유연성 측면에서 강점을 가지며, 자동화와 재생에너지 연계가 핵심 기술로 강조됨

<br/>

# Methodology
- **Simulation:** Flownex SE를 활용한 i-SMR 4개 모듈 열유체 시스템 모델링 (170MWe 수준)
- **RL Framework:** Gymnasium 패키지를 활용한 강화학습 환경 구축
- **Algorithm:** SAC(Soft Actor-Critic) - 엔트로피 개념을 도입하여 탐험과 활용의 균형을 맞추는 Off-Policy Actor-Critic 알고리즘
- **State:** 44개 연속 변수 (모듈당 11개 상태 × 4개 모듈)
- **Action:** 12개 연속 변수 (제어봉 삽입도, 메인밸브 개도율, 주급수 펌프 RPM × 4개 모듈)
- **Reward Function:** 출력 추종 오차, 감발량 분배 균형, 종료 이벤트 패널티를 고려한 가중합 설계

<br/>

# Key Results
- 기상청 신재생에너지 발전량 시뮬레이션을 활용한 목표 출력 설정
- 4개 모듈의 초기 연소도를 각각 10%, 50%, 70%, 90%로 설정하여 실험
- 1,000회 에피소드 학습을 통해 수렴 경향 및 성능 향상 추세 확인
- 단일 정책으로 모듈 간 상호작용을 반영한 운전 시나리오 수행
- 재생에너지 변동 대응 가능성 검증
